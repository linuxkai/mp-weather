# mp-weather
天气预报微信小程序

## 说明
自己看官方文档写的第一个微信小程序
UI：参考了华为的自带的天气预报页面
数据来源：使用的是和风天气的接口,文档: https://dev.qweather.com/docs/
天气ICON：来自和风天气Icon(https://icons.qweather.com/)
